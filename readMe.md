# flaskAPI_horse

## Description

Educational project for learning Flask and API.

## Hosted
[Vercel](https://api-horse.vercel.app/)

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install Flask.

```bash
pip install requirements.txt
```

## Usage

```bash
python app.py
```
